# 아마노코리아 브랜드 디자인 시스템 적용 완료 🎨

## 📋 개요
아마노코리아 PPT 자동화 시스템의 모든 페이지를 **브랜드 컬러**와 **현대적인 디자인**으로 통일했습니다.

---

## 🎨 디자인 시스템

### 브랜드 컬러
- **주 색상**: `#0081C0` (아마노코리아 블루)
- **보조 색상**: `#005A8D` (진한 블루)
- **강조 색상**: `#0095D9` (밝은 블루)
- **텍스트**: `#003D5C` (진한 블루 - 시인성 우수)

### 배경 그라디언트
```css
background: linear-gradient(135deg, #f5f7fa 0%, #e8f0f7 50%, #d5e5f2 100%)
```

### 디자인 원칙
1. **Glassmorphism** - 유리 질감 효과
2. **고대비 텍스트** - 뛰어난 시인성
3. **부드러운 애니메이션** - 자연스러운 전환
4. **일관된 스타일** - 모든 페이지 통일

**주요 개선 사항:**
- ✨ Glassmorphism 카드 (35% 불투명도)
- 🌈 애니메이션 그라디언트 배경 (브랜드 컬러)
- 📖 진한 텍스트 색상 (#003D5C)
- 💎 로그인 버튼 브랜드 컬러 그라디언트

- ✨ 떠다니는 배경 요소

**특징:**

### 색상 시스템
| 요소 | 색상 | 용도 |
|------|------|------|
| 배경 그라디언트 | #f5f7fa → #d5e5f2 | 모든 페이지 배경 |
| 브랜드 컬러 | #0081C0 | 헤더, 버튼, 아이콘 |
| 진한 블루 | #005A8D | 그라디언트 보조 |
| 텍스트 | #003D5C | 입력 필드, 레이블 |
| 흰색 | #FFFFFF | 카드, 버튼 배경 |

### 타이포그래피
- **제목**: 600-800 (SemiBold-ExtraBold)
- **본문**: 400-600 (Regular-SemiBold)
- **캡션**: 400-500 (Regular-Medium)

### 간격 시스템
- **작은 간격**: 8px (0.5rem)
- **기본 간격**: 16px (1rem)
- **큰 간격**: 24-32px (1.5-2rem)
- **컨테이너 패딩**: 32px (2rem)

### 그림자 시스템
- **카드**: `0 8px 32px rgba(0, 0, 0, 0.1)`
- **호버**: `0 12px 48px rgba(0, 0, 0, 0.15)`
- **Glassmorphism**: `0 8px 32px rgba(0, 129, 192, 0.2)`
