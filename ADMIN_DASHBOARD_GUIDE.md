## 🔍 대시보드 구조

### 📊 **업무 현황 탭** (기본 화면)

#### 1️⃣ **전체 통계 카드 (6개)**

```
┌─────────────┬─────────────┬─────────────┬─────────────┬─────────────┬─────────────┐
│ 오늘 마감   │ 이번주      │ 긴급 (D-3)  │ 예정        │ 진행중      │ 완료        │
│ 🔵 숫자     │ 🟣 숫자     │ 🔴 숫자     │ 🟡 숫자     │ 🔵 숫자     │ 🟢 숫자     │
└─────────────┴─────────────┴─────────────┴─────────────┴─────────────┴─────────────┘
```

**각 카드의 의미:**
- **오늘 마감**: 오늘 마감 예정인 업무 개수
- **이번주**: 이번 주 내 마감되는 업무 개수 (일요일~토요일)
- **긴급 (D-3)**: 마감일까지 3일 이하 남은 업무 개수
- **예정**: Todo 상태 업무 개수
- **진행중**: Doing 상태 업무 개수
- **완료**: Done 상태 업무 개수

#### 2️⃣ **팀원별 업무 현황 카드**

각 팀원마다 개별 카드가 표시됩니다:

```
┌─────────────────────────────────────┐
│ 홍세영 (계장)              총 업무: 8 │
├─────────────────────────────────────┤
│ [예정: 3] [진행중: 2] [완료: 3]    │
├─────────────────────────────────────┤
│ 🔥 오늘 마감 (2개)                  │
│ • 고객사 제안서 작성                 │
│ • 마케팅 자료 업데이트               │
├─────────────────────────────────────┤
│ ⚠️ 긴급 (D-3) (1개)                │
│ • 사업 계획서 초안                   │
└─────────────────────────────────────┘
```

**카드 구성 요소:**
- **헤더**: 팀원 이름 + 직급 + 총 업무 개수
- **상태별 통계**: 예정(노란색) / 진행중(파란색) / 완료(초록색)
- **오늘 마감 섹션**: 오늘 마감되는 업무 목록 (빨간색 강조)
- **긴급 업무 섹션**: D-3 이하 업무 목록 (노란색 강조)

#### 3️⃣ **미배정 업무 섹션**

배정되지 않은 업무들을 한눈에 확인:

```
┌─────────────────────────────────────────────────────────┐
│ 미배정 업무 (5개)                                        │
├─────────────────────────────────────────────────────────┤
│ • 고객사 PPT 제작 | 디자인 | 영업부                     │
│   마감: 2026-01-25   [🔴 오늘 마감]                     │
├─────────────────────────────────────────────────────────┤
│ • 사내 행사 포스터 | 디자인 | 총무부                    │
│   마감: 2026-01-28   [🟡 D-3]                           │
└─────────────────────────────────────────────────────────┘
```

### 📋 **업무 배정 탭** (Drag & Drop)

#### Kanban 보드 형식

```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│ 미배정 업무 │ 홍세영 (계장)│ 최예지 (사원)│ 홍두의 (사원)│
│             │             │             │             │
│ [업무카드1] │ [업무카드]  │ [업무카드]  │ [업무카드]  │
│ [업무카드2] │ [업무카드]  │             │             │
│             │ [업무카드]  │             │             │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

**사용 방법:**
1. 미배정 업무 카드를 드래그
2. 원하는 팀원 컬럼에 드롭
3. 자동으로 업무가 배정되고 상태가 'Todo'로 변경

**역배정:**
- 팀원의 업무를 "미배정" 컬럼으로 드래그하면 배정 해제

## 🚀 주요 기능

### 1️⃣ **실시간 통계 확인**

- 새로고침 버튼(🔄)으로 최신 데이터 업데이트
- 팀 전체 업무 진행률 한눈에 파악
- 팀원별 업무 부하 확인

- Drag & Drop으로 빠른 업무 배정
- 팀원별 현재 업무량 확인 후 배정
- 긴급 업무 우선 배정 가능

### 3️⃣ **위험 업무 사전 파악**

- 오늘 마감 업무 자동 강조
- D-3 이하 긴급 업무 별도 섹션
- 지연된 업무 빨간색 표시

### 4️⃣ **주간 보고서 자동 생성**

- 하단의 "📊 주간보고서 PPT 생성" 버튼 클릭
- 완료된 업무 자동 수집
- PPT 파일 자동 다운로드

## 💡 활용 팁

### **매일 오전 루틴**
1. 로그인 후 대시보드 확인
2. "오늘 마감" 카드 숫자 확인
3. 팀원별 "오늘 마감" 섹션 확인
4. 필요시 Slack/이메일로 리마인드

### **업무 배정 시**
1. "업무 배정" 탭으로 전환
2. 미배정 업무 확인
3. 각 팀원의 현재 업무량(Todo/Doing/Done) 확인
4. 업무 특성과 팀원 전문성 고려
5. Drag & Drop으로 즉시 배정

### **주간 회의 준비**
1. "업무 현황" 탭에서 전체 통계 확인
2. 팀원별 완료 업무 개수 확인
3. "주간보고서 PPT 생성" 버튼 클릭
4. 생성된 PPT로 회의 진행

### **위험 관리**
- 🔴 **지연** 배지가 있는 업무 즉시 확인
- 🟡 **D-3** 이하 업무는 진행 상태 점검
- 한 팀원에게 업무가 집중되면 재배분 검토

## ⚙️ 기술 정보

### **사용 기술**
- **Frontend**: Next.js 14 + TypeScript
- **Database**: Supabase (PostgreSQL)
- **Drag & Drop**: @hello-pangea/dnd
- **Styling**: Tailwind CSS

### **주요 API**
- `GET /api/tasks` - 업무 목록 조회
- `GET /api/profiles` - 팀원 목록 조회
- `PATCH /api/tasks/:id` - 업무 배정/상태 변경
- `POST /api/pptx/generate` - PPT 생성

## 🆘 문제 해결

### **팀원이 표시되지 않는 경우**
1. Supabase에서 사용자 계정 확인
2. `public.profiles` 테이블에 프로필 존재 확인
3. `role = 'member'` 설정 확인

### **업무가 표시되지 않는 경우**
1. "🔄 새로고침" 버튼 클릭
2. 브라우저 콘솔(F12) 에러 확인
3. `/request` 페이지에서 새 업무 생성 테스트

### **Drag & Drop이 작동하지 않는 경우**
1. 브라우저 캐시 삭제
2. 페이지 새로고침
3. 다른 브라우저로 테스트 (Chrome 권장)
